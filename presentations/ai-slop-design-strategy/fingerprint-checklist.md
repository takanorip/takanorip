# AIスロップ指紋チェックリスト

**用途**: AI生成またはAI支援で作ったUI/UIコードの出荷前レビュー  
**所要時間**: 約5分  
**判定**: 1項目でも該当すれば要修正。3項目以上で「スロップ確定」

---

## タイポグラフィ

- [ ] 主フォントが Inter / Roboto / Arial / system-ui のいずれか
- [ ] Display と Body で同じフォントファミリーのみ
- [ ] 見出しにグラデーション文字（`bg-clip-text` 等）を使用
- [ ] フォントサイズが Tailwind デフォルト刻みのみ（微調整なし）

## カラー

- [ ] 主色が紫 / インディゴ / バイオレット系
- [ ] ヒーロー背景が `#0f0f23` 系の暗色 + 紫グラデーション
- [ ] 装飾目的のグラデーション（意味のない背景グラデ）
- [ ] CTAボタンのコントラスト比が 4.5:1 未満

## レイアウト

- [ ] ヒーロー → 3等分機能カード → CTA のみの構成
- [ ] 全セクションが同じ max-width / padding
- [ ] カードがすべて同一サイズ・同一構造

## 装飾

- [ ] ガラスモーフィズム（`backdrop-blur` + 半透明背景）
- [ ] グロー影（`box-shadow` with color spread）
- [ ] border + drop-shadow の二重装飾
- [ ] 角丸 16px 以上が小カードに一律適用

## アイコン・画像

- [ ] 絵文字（✨🚀🔒等）が機能アイコン代わり
- [ ] Lucide / Heroicons を無加工で使用
- [ ] ストック写真または抽象的なAI生成イラストのみ

## コピー

- [ ] 「革新的」「次世代」「シームレスな体験」等の抽象ワード
- [ ] 機能説明が3項目に機械的に分割
- [ ] 固有名詞・具体数値・実スクショへの言及なし

## モーション

- [ ] bounce / elastic イージングの多用
- [ ] スクロール連動アニメーションの過剰使用
- [ ] すべての要素に hover アニメーション

## プロセス

- [ ] DESIGN.md または同等の仕様書が存在しない
- [ ] 1案のみで方向性の比較をしていない
- [ ] 戦略ブリーフ（形容詞・禁止形容詞）が未作成
- [ ] レンダリング結果の計測検証をしていない

---

## 判定基準

| 該当数 | 判定 | アクション |
|-------|------|-----------|
| 0 | ✅ Pass | 出荷可 |
| 1〜2 | ⚠️ Warning | 該当項目を修正 |
| 3+ | ❌ Slop | 方向性から再検討 |

---

## 修正の優先順位

1. **フォントを変える** — 最も効果が高い
2. **主色をブランド色に固定** — グラデーションを外す
3. **実物のスクショ・写真を入れる**
4. **コピーを人間が書き直す**
5. **DESIGN.md を作成して再生成**
